你这段代码里大约 **20 个参数**，但真正值得做实验分析的通常只有 **6–9 个**。

---

# 一、最关键参数（必须分析）

这些参数 **直接影响模型表达能力或学习过程**。

### 1 `lr`（learning rate）

```python
--lr
```

作用：

* 控制梯度更新步长

典型取值：

```
0.01
0.005
0.001
0.0005
0.0001
```

影响：

* 太大 → 不收敛
* 太小 → 收敛慢

**几乎所有论文都会分析**

---

### 2 `latdim`（embedding dimension）

```python
--latdim
```

作用：

```
节点 embedding 维度
```

影响：

```
模型表达能力
```

典型取值：

```
16
32
64
128
256
```

影响趋势：

```
太小 → 表达不足
太大 → 过拟合 + 计算慢
```

推荐做箱线图。

---

### 3 `gnn_layer`

```python
--gnn_layer
```

作用：

```
GNN传播层数
```

含义：

```
1层 = 一阶邻居
2层 = 二阶邻居
3层 = 三阶邻居
```

典型取值：

```
1
2
3
4
```

影响：

```
太深 → over-smoothing
```

GNN论文 **必分析参数之一**。

---

### 4 `num_head`

```python
--num_head
```

作用：

```
Transformer multi-head attention 数量
```

典型取值：

```
1
2
4
8
```

影响：

```
注意力表达能力
```

---

### 5 `att_head`

```python
--att_head
```

作用：

```
attention heads
```

如果这个模型有 **两个 attention 模块**
那就需要一起分析。

---

### 6 `dropout`

```python
--dropout
```

作用：

```
防止过拟合
```

典型取值：

```
0
0.1
0.2
0.3
0.5
```

# 五、箱线图应该怎么做

实验流程：

```
固定其他参数
改变一个参数
训练模型
记录指标
```

例如：

```
lr = [0.01,0.005,0.001,0.0005]
```

每个训练 **5次**

记录：

```
Recall@20
NDCG@20
```

得到：

```
20个结果
```

画：

```
箱线图
```

---

### Python画图示例

```python
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

data = {
    "lr=0.01":[0.41,0.42,0.39,0.40,0.43],
    "lr=0.005":[0.45,0.46,0.44,0.47,0.45],
    "lr=0.001":[0.50,0.49,0.51,0.50,0.52],
    "lr=0.0005":[0.47,0.48,0.46,0.47,0.49],
}

df = pd.DataFrame(data)

sns.boxplot(data=df)

plt.ylabel("Recall@20")
plt.show()
```

# 七、一个很重要的建议（很多人做错）

不要同时改多个参数。

正确方法：

```
ablation study
```

一次只改：

```
一个参数
```

否则：

```
实验没有解释性
```
